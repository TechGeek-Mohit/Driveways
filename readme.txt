Useful git commands:


New Branch:

git pull

git checkout -b "new-branch-name"

git add "filename"





Clone Repo:
	git clone https://github.com/michmoosch/CS157A-team1.git

Change branch:
	git checkout <branch-name> 
	(To see current branch use: git checkout)

Check status:
	git status

Check username:
	git config user.name

Add files to staging:
	git add "filename.txt"

Commit changes:
	git commit -m "Your message goes here"

Push branch changes to remote repo: 
	git push
	git push --set-upstream <branch-name> if your branch doesn't exist in the remote repo
